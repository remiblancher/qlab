CODE SIGNING WORKFLOW
---------------------

Same workflow, different algorithm:

Classical                              Post-Quantum
─────────                              ────────────

1. Generate key pair                   1. Generate key pair
   (ECDSA P-384)                          (ML-DSA-65)

2. Get code signing cert               2. Get code signing cert
   from CA                                from CA

3. Sign binary                         3. Sign binary
   signcode.exe -sign ...                 signcode.exe -sign ...

4. Distribute                          4. Distribute

5. User verifies                       5. User verifies
   (signature valid)                      (signature valid)


     VULNERABLE                             QUANTUM-SAFE
     to future quantum                      forever
     attack
